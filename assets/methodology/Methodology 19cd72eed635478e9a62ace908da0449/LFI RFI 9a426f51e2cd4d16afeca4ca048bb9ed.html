LFI/RFI

  • File inclusion requires the scripting engine to execute the file being referenced
  • LFI/RFI is different than directory traversal
    • LFI/RFI requires inclusion and execution of files being traversed
  • LFI allows us to include files on the servers local filesystem

LFI Example:

http://example.com/index.php?page=/etc/passwd
