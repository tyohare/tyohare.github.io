Regular Penetration testing process:

  • Reconnaissance
    • Active and passive
  • Scanning/Enumeration
  • Exploitation
  • Privilege Escalation
  • Persistence/Covering Tracks
  • Reporting

Mobile Pentesting Process:

  • recon
    • Read Reviews
    • find in on app store and read review or patch notes for issues
    • enumerate other apps by the company
  • Static analysis
    • manual and automated source code review
  • Dynamic analysis
    • Burp suite
    • OWASP top 10/Mobile top 10
