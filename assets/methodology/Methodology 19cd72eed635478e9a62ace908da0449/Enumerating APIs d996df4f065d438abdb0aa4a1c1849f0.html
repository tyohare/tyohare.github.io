Enumerating APIs

Fuzzing APIs:

  • We can fuzz APIs using Burp Suite Intruder or wfuzz.
wfuzz -c -z file,/usr/share/wordlists/dirb/common.txt --sc 200 'http://0.0.0.0:5000/api/v1/resources/books?show=FUZZ' 

ffuf -u http://localhost/v1/FUZZ -w /usr/share/wordlists/dirb/big.txt -e .php

Discovery via Source Code:

  • we can run through the application using Burp and capture requests for all the functionality
  • View JS files in the website. (Use JS formatter to make the code look pretty).
