DNS (53)

Forward Lookup:

for ip in $(cat list.txt); do host $ip.megacorpone.com; done

Reverse Lookup:

for ip in $(seq 50 100); do host 0.0.0.$ip; done | grep -v "not found”
for ip in $(seq 1 225); do host -l 0.0.0.0 0.0.0.$ip; done | grep -v "failed:”

Others:

host -l [test.com](http://test.com/) [ns1.test.com](http://ns1.test.com/)

host -t ns [test.com](http://test.com/) | cut -d " " -f 4

dnsrecon -d [test.com](http://test.com/) -t axfr

dnsenum [zonetransfer.me](http://zonetransfer.me/)

dig -axfr IP

Example:

First

$ nslookup 10.10.10.13 10.10.10.13
13.10.10.10.in-addr.arpa	name = ns1.cronos.htb.

Then:

dig axfr cronos.htb @10.10.10.13

We find additional domains to add to /etc/hosts
