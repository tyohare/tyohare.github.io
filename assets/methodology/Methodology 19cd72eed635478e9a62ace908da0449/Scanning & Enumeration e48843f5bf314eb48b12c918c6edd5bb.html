Scanning & Enumeration

  • arp-scan -l
  • netdiscover
  • use arp to discover machines on the network

  • Under 64 on a ping is high probability Linux
  • Under 128 is high probability Windows

Host discovery Scripts:

sudo nmap -sU --top-ports 100 $target

Port scanning scripts:

nmap -sC -sV -p 135,445 0.0.0.0 --script=safe

nmap -script=smb-vuln\* -p445 0.0.0.0

Nmap:

nmap -sS (stealth scan, sends SYN with RST flag so connection doesnt actually connect)

nmap -T4 -p- -A <ip address>

-T5 = speed of 5

-p- = all ports scan otherwise scans top 1000 ports

-A = All everything scripts scans etc

-sU = udp scan

-sV = enumerate Version

-sT = TCP connect

-sC = default enumeration scripts

-pN = dont worry about pinging

-script vuln = vulnerability detection scripts (safe?)

Nmap with nessus scan, then while scanning go ahead and do information gathering stuff
