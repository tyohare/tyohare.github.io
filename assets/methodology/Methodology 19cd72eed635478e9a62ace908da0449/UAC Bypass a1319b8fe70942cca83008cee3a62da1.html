UAC Bypass

  • Once you have a reverse shell, check the user permissions with
  • whoami /groups
  • Even though you're a local admin, you'll see the shell you have is running at a Medium Mandatory Level. That means that many actions will be prohibited by UAC, and this should ring a bell.
  • Because of the way of how UAC exploit work, they are often developed for specific OS versions and build.
  • systeminfo
  • To get more information about that (OS Name and OS Version)

    - Google "OS name + version + 'uac bypass'".

Disable UAC:

C:\Windows\System32\cmd.exe /k %windir%\System32\reg.exe ADD HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\Policies\System /v EnableLUA /t REG_DWORD /d 0 /f
